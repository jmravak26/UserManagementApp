CREATE REACT PROJECT (without TS):
npm create vite@latest <name> --template react
cd <name>

OR

CREATE REACT PROJECT (with TS):
npm create vite@latest <name> --template react-ts
cd <name>

INSTALL DEPENDENCIES:
npm install

npm run dev
