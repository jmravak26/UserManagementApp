# User Management Application

A full-stack user management system built with React frontend and backend API.

## Project Structure

```
managementApp/
├── user-management-app/    # React frontend application
├── backend/               # Backend API server
└── README.md             # This file
```

## Getting Started

### Frontend (React App)
```bash
cd user-management-app
npm install
npm run dev
```

### Backend (API Server)
```bash
cd backend
# Setup instructions will be added when backend is implemented
```

## Features

- User CRUD operations
- Role-based permissions (Admin, Manager, User)
- Bulk operations and data export
- Advanced filtering and sorting
- User profile management

## Development

This is a monorepo containing both frontend and backend code. Each directory has its own package.json and can be developed independently.
